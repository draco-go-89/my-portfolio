# 📦 Build Artifacts

This directory contains compiled **Android APK** builds for the mobile applications showcased in the portfolio.

## Files

| File | App | Version | Description |
|------|-----|---------|-------------|
| `Lubo_v4.apk` | Music Player V2 | v4 | Simple music player with enhanced UI |
| `musicplayer1.apk` | Music Player App | v1 | Music player with password protection login |
| `todolist.apk` | To Do List App | — | Smart task manager with alarm system |

## Installation

### Android Device
1. Download the desired `.apk` file to your Android device.
2. Open the file from your Downloads folder.
3. If prompted, allow installation from **Unknown Sources** in Settings → Security.
4. Follow the on-screen instructions to install.

### Emulator
```bash
# Using Android Debug Bridge (ADB)
adb install Lubo_v4.apk
adb install musicplayer1.apk
adb install todolist.apk
```
